# Contributing to CIPHER Trading Skill

First off — thank you for considering a contribution! 🙌

This is a community resource. Every improvement makes CIPHER better for all traders using it.

---

## 📋 Ways to Contribute

### 🐛 Report Issues
- Found an error in a reference file?
- Signal format has a bug or missing field?
- Open an [Issue](../../issues) with a clear description

### 📈 Improve the Skill
- Better signal card format
- Additional market coverage (new assets/pairs)
- New reference file (e.g., options trading, futures basis)
- Language improvements (Urdu/other language versions)

### 📚 Add Reference Content
- Expand existing reference files with more patterns or indicators
- Add accuracy/reliability data to candlestick patterns
- Improve position sizing formulas

---

## 🛠️ How to Submit a PR

1. **Fork** this repository
2. **Clone** your fork locally
   ```bash
   git clone https://github.com/YOUR_USERNAME/cipher-trading-skill.git
   ```
3. **Create a branch** for your change
   ```bash
   git checkout -b improve/your-feature-name
   ```
4. **Make your changes** — keep them focused and clear
5. **Commit** with a meaningful message
   ```bash
   git commit -m "Add: 3 new candlestick patterns to reference file"
   ```
6. **Push** to your fork
   ```bash
   git push origin improve/your-feature-name
   ```
7. Open a **Pull Request** — describe what you changed and why

---

## ✅ Guidelines

- Keep the CIPHER persona consistent — authoritative, precise, no hype
- All signal formats must include Stop Loss (non-negotiable)
- Reference files should cite reliable sources where possible
- Do not add financial advice — always keep educational framing
- English for signal cards; other languages for explanations only
