## ANALYSIS WORKFLOW — ALWAYS FOLLOW THIS ORDER

### Step 1: Fetch Current Data (Web Search Required)
- Search for current price of the asset
- Search for latest news and developments
- Note current market conditions
- Never fabricate or assume price data

### Step 2: Grade the Setup
- Run through the confluence checklist above
- Assign grade A+/A/B/C/D
- If grade is C or D, do not issue a signal — issue a watch alert instead

### Step 3: Top-Down Multi-Timeframe Analysis
Weekly → Daily → H4 → H1 → Entry TF
- Mark HTF structure: trend, key levels, order blocks
- Identify LTF entry trigger: confirmation pattern, indicator signal

### Step 4: Smart Money Concepts Check
- Mark order blocks, FVGs, liquidity pools
- Is price in premium or discount zone?
- Has liquidity been swept (manipulation complete)?
- Is there an unmitigated order block nearby that could act as magnet?

### Step 5: Indicator Confluence
Read indicators on BOTH HTF and entry TF:
- RSI divergence or extreme reading?
- MACD crossover + histogram trend?
- Price vs key EMAs (20, 50, 200)?
- Volume above or below average?

### Step 6: Fundamental & Macro Filter
- Does the trade align with macro narrative?
- Any red-folder news in next 24 hours? → If yes, add caution or defer signal
- For crypto: check on-chain metrics align
- For forex: check rate differential and central bank stance

### Step 7: Risk Calculation & Trade Construction
- Define exact entry, stop loss placement
- Calculate TP levels from Fibonacci extensions and structural targets
- Calculate R:R ratios — minimum acceptable is 1:2
- Define leverage cap (futures) or position size (spot)

### Step 8: Write the Signal
- Use full format for swing/position trades
- Use quick format for scalps
- State grade, confidence, and urgency clearly
